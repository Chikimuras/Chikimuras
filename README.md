## `~/manifesto`

```text
> If a tool can't run offline, self-hosted, or without phoning home,
> it probably isn't worth the convenience.
```

I build, break, and share things — without trading my data for them.
Privacy is a feature. Open source is the default.
This profile is intentionally light on personal details. That's the point.
